# 🚀 NextStep PRD (Final / Dynamic AI Routing)

## 1. 프로젝트 개요

**프로젝트명:** NextStep  

**목표:**  
사용자가 원하는 직무를 입력하면,  
AI가 해당 직무를 해석하고 개인의 현재 상태를 반영하여  
최적의 학습 경로를 생성하는 교육 라우팅 시스템 구축

---

## 2. 문제 정의

### 2.1 학습자 문제
- 무엇을 어떤 순서로 학습해야 하는지 알기 어려움
- 현재 학습과 목표 직무 간의 연결성이 보이지 않음
- 개인 수준이 반영되지 않은 커리큘럼

### 2.2 교육 시스템 문제
- 정적인 커리큘럼 구조로 인해 유연성이 부족함
- 새로운 기술 반영이 어려움
- 다양한 직무를 커버하기 어려움

---

## 3. 핵심 개념

NextStep은 학습을 “강의 추천”이 아닌  
**“목표 직무까지의 경로 생성 문제”로 정의한다.**

이 시스템은 특정 직무를 미리 정의하지 않고,  
AI가 사용자의 입력을 기반으로 직무를 해석하고  
그에 맞는 학습 경로를 동적으로 생성한다.

---

## 4. AI 중심 구조

AI는 단순 보조 기능이 아니라  
**시스템의 핵심 엔진 역할을 수행한다.**

---

### 4.1 직무 동적 생성 (Job Generation)

- 사용자가 직무를 자연어로 입력한다  
- AI가 해당 직무를 해석한다  
- 필요한 기술 목록과 중요도를 도출한다  

👉 특정 직무를 미리 정의하지 않아도 모든 직무 대응 가능

---

### 4.2 사용자 상태 해석 (User Understanding)

- 사용자가 자신의 학습 상태를 입력한다  
- AI가 이를 분석하여 현재 수준을 판단한다  
- 학습 시작 지점을 결정한다  

---

### 4.3 기술 관계 보완 (Graph Augmentation)

- 기존 기술 간 관계가 부족한 경우  
- AI가 기술 간 연결을 추론하여 보완한다  

---

### 4.4 학습 경로 생성 지원

AI는 다음 요소를 고려하여  
경로 생성에 필요한 정보를 제공한다:

- 기술 중요도
- 학습 난이도
- 시장 수요
- 선수 관계

---

### 4.5 추천 이유 생성 (Explainability)

각 학습 단계에 대해 다음을 생성한다:

- 해당 기술이 필요한 이유
- 직무 내 활용도
- 다음 단계와의 연결성

---

## 5. 주요 기능

---

### 5.1 목표 기반 경로 생성

- 사용자가 직무를 입력한다  
- 시스템이 해당 직무에 맞는 학습 경로를 생성한다  

---

### 5.2 개인화된 출발점 설정

- 사용자 상태를 기반으로  
- 서로 다른 시작 지점에서 경로를 생성한다  

---

### 5.3 최적 학습 경로 제공

경로 생성 시 다음 요소를 고려한다:

- 학습 난이도  
- 직무 중요도  
- 시장 수요  
- 선수 학습 필요성  

---

### 5.4 학습 경로 시각화

- 전체 학습 흐름을 시각적으로 제공  
- 현재 위치와 목표를 명확히 표현  

---

### 5.5 추천 이유 제공

각 학습 단계에 대해 설명 제공:

- 왜 필요한지  
- 어떤 역할을 하는지  
- 어떻게 활용되는지  

---

### 5.6 다중 경로 제공

하나의 직무에 대해 다양한 경로 제공:

- 빠른 취업 중심 경로  
- 기본기 중심 경로  

---

### 5.7 콘텐츠 연계

각 기술 단계는 실제 학습 콘텐츠와 연결된다.

---

## 10. UX / 디자인 정의

### 디자인 컨셉

> **"전체 지도 위에서, 나에게 필요한 길만 강조되는 구조"**

- **Dark Mode + Glassmorphism** 기반 UI
- 다크 배경 위 그래디언트 오브(deep purple, neon blue, teal)
- 노드 카드: `backdrop-filter: blur(12px)`, 반투명 배경, 미세한 보더 글로우
- 게이미피케이션 요소를 적용한 스킬트리 느낌

---

### 10.1 전체 그래프 + 추천 경로 강조

- 전체 기술 맵은 흐릿하게(dimmed) 표시된다
- 추천된 학습 경로만 밝은 엣지와 글로우 효과로 강조된다
- 강조 경로의 엣지는 애니메이션 흐름(pulse)으로 방향성을 표현한다

**효과:** 전체 구조가 보이면서 내가 가야 할 길이 명확해진다

---

### 10.2 현재 위치 표시

- 사용자의 현재 학습 위치가 그래프 상에서 명확히 표시된다
- 현재 노드: pulsing ring 애니메이션 + 강조 색상
- 주변 노드와의 연결선이 시각적으로 구분된다

**효과:** "내가 어디 있는지" 즉시 인지 가능

---

### 10.3 목표 지점 강조

- 목표 직무에 해당하는 최종 노드는 시각적으로 강조된다
- 목표 노드: 별도 아이콘 + 글로우 이펙트 + 크기 확대

**효과:** 목적지 인식, 동기 부여

---

### 10.4 분기 구조 표현

하나의 목표에 대해 여러 학습 경로가 분기 형태로 제공된다:

- **빠른 취업 루트**: 핵심 기술만 최단 경로로
- **기본기 루트**: 기초부터 탄탄하게

각 루트는 색상/스타일로 구분되며, 사용자가 경로를 전환할 수 있다

**효과:** 선택 가능성 제공, 사용자 주도 학습

---

### 10.5 노드 인터랙션

사용자가 노드를 클릭하면 슬라이드 패널로 상세 정보가 표시된다:

- 기술 설명
- 추천 이유 (AI 생성)
- 연결된 학습 콘텐츠
- 예상 학습 시간
- 선수 학습 요건

**효과:** 각 단계의 맥락 이해, 학습 동기 부여

---

### 10.6 진행 상태 표현

이미 학습한 노드와 앞으로 학습할 노드가 시각적으로 구분된다:

| 상태 | 시각 표현 |
|------|----------|
| 완료 | 체크마크 + 채워진 색상 |
| 진행 중 | 펄싱 링 애니메이션 |
| 미완료 | 흐림(dimmed) + 잠금 아이콘 |
| 추천 다음 단계 | 미세 글로우 + 화살표 |

**효과:** 성취감 + 다음 할 일 명확

---

### 10.7 반응형 전략

- **데스크탑**: 자유 팬/줌 + 미니맵(React Flow 내장) + 사이드 상세 패널
- **모바일**: 스크롤 기반 세로 경로 뷰 + 노드 탭 시 바텀시트 패널
- 모바일에서는 Framer Motion으로 노드 순차 등장 애니메이션 적용

## 11. 확장 방향

- 기술 그래프 고도화  
- 추천 정확도 개선  
- 콘텐츠 다양화  
- 기업 및 교육기관 대상 서비스 확장  

---

## 12. 사용자 흐름

1. 사용자가 목표 직무를 입력한다  
2. 사용자가 현재 상태를 입력한다  
3. AI가 직무와 상태를 해석한다  
4. 학습 경로가 생성된다  
5. 사용자가 시각화된 경로를 확인한다  
6. 각 단계의 설명을 통해 학습 이유를 이해한다  
7. 필요 시 다른 경로를 선택한다  

---

## 13. 성공 기준

- 어떤 직무 입력에도 경로 생성이 가능해야 한다  
- 사용자 상태에 따라 경로가 달라져야 한다  
- 각 학습 단계에 대한 설명이 제공되어야 한다  
- 신규 기술 추가 시 경로에 반영되어야 한다  

---

## 14. 핵심 가치

NextStep은  
AI를 통해 직무를 해석하고 학습 경로를 생성함으로써,  
사용자의 현재 위치에서 목표까지의 가장 현실적인 학습 흐름을 제공하는  
교육 내비게이션 시스템이다.
